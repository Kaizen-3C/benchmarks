## Conventions for the skeletons

Each metric module exports one function:
```python
def compute(original_dir: Path, recomposed_dir: Path, **kwargs) -> dict:
    """Return {'metric': 'q1_test_parity', 'value': float|None, 'detail': dict}."""
    return {"metric": "q1_test_parity", "value": None, "detail": {"todo": "phase 2"}}
```

Each gate module exports one function:
```python
def check(spec_dir: Path, original_dir: Path) -> dict:
    """Return {'gate': '<name>', 'pass': bool|None, 'failures': list[dict]}."""
    return {"gate": "coverage", "pass": None, "failures": []}
```
